# FlatStore

FlatStore is a modern application store built with React and Electron. It provides a clean and intuitive interface for discovering and managing applications.
(this is a wrapper for the entire flubhub repo so you dont have to visit the site)
## Features

- **App Discovery:** Browse a curated list of applications.
- **Search:** Quickly find specific apps with the built-in search functionality.
- **App Details:** Detailed information, screenshots, and metadata for each application.
- **Modern UI:** Built with React and Tailwind CSS for a responsive and sleek experience.
- **Electron Powered:** Runs as a desktop application with system-level capabilities.

## Tech Stack

- **Frontend:** [React](https://reactjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Desktop Environment:** [Electron](https://www.electronjs.org/)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/reefer42/flatstore.git
   cd flatstore
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

To start the application in development mode:
   ```bash
   npm run dev
   ```

### Building

To build the application for production:
   ```bash
   npm run build
   ```
